# Cendekia — Landing Page (Quiet Luxury)

Landing page mewah-minimalis untuk "Asisten Riset Dokumen ber-AI".

## Cara pakai
1. Ekstrak folder ini.
2. Buka `index.html` di browser (klik dua kali).
3. Ganti `ISI-LINK-STREAMLIT-MU` di `index.html` (2 tempat) dengan URL aplikasi Streamlit-mu.

## Font
- **Fraunces** (serif editorial) — judul
- **Inter** — teks
- **Space Mono** — label teknis

## Animasi & interaksi
- **Preloader Garuda** (pembuka)
- **Demo chat AI langsung** di hero (tanya → mengetik → jawab + sitasi) — ganti medali lama
- **Animasi mengetik** baris hero
- **Debu emas** melayang di background
- **Latar berubah warna + aurora bergerak** tiap bagian
- **Reveal scroll beragam** (naik / kiri / kanan / skala / miring 3D) — dua arah
- **Teks scramble** pada label bagian
- **Judul mask-reveal**, statistik menghitung, kursor kustom, tombol magnetik, kartu miring 3D, bar progres
- Semua hormat `prefers-reduced-motion` (mode hemat animasi otomatis)

## Warna (ubah di `:root` — `style.css`)
- `--gold #d9a441` (aksen emas)
- Warna latar per bagian ada di atribut `data-bg1/--bg2/--acc` tiap `<section>` di `index.html`

## Statistik
Angka di bagian "Dipercaya" sudah jujur (bukan pengguna palsu). Sesuaikan bila perlu.
